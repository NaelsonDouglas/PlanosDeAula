# Planos de Aula

Repositório de planos de aula organizados por disciplina e turma.

---

## Estrutura

```
disciplina/
  turma/
    aulas/       — planos de aula em Markdown
    aulas_html/  — versão HTML para visualização
```

---

## Criar um novo plano de aula

Para gerar um novo conjunto de planos, envie as informações abaixo:

```
Professora:
Disciplina:
Turma: (ex: 1-01)
Turno: (M / V / I / M/V)
Escola: (Ozória / Fernandina)

Período letivo:
  Início:
  Fim:
  Recesso (se houver): de ___ até ___

Dia da semana das aulas:
Duração de cada aula: (ex: 1H)

Tópicos em ordem:
1.
2.
3.
```
